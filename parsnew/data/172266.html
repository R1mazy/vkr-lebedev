Pandas, Numpy, Matplotlib - для начинающих.

Этот курс для тех, кто хочет быстро начать учиться и использовать эти библиотеки. В этом курсе вы узнаете о незаменимых библиотеках Python и научитесь использовать Pandas для работы с данными, Numpy для быстрых научных вычислений и Matplotlib для визуализации данных. Этот курс является лишь введением для новичков…
Beginner Level
2-3 часа в неделю

59 learners

What you will learn

  • Применять Python для анализа и визуализации данных из разных источников и в разных форматах.
  • Строить информативные и красивые графики и диаграммы с помощью Matplotlib.
  • Использовать визуализацию данных для поддержки своих выводов и рекомендаций.
  • Интегрировать визуализацию данных в свои профессиональные проекты и презентации.
  • загружать данные из различных источников (например, CSV-файлов).
  • выполнять различные операции с данными (например, фильтрацию).
  • выполнять агрегирование данных.
  • выполнять математические операции над массивами данных (например, сложение и умножение).
  • создавать различные типы графиков (например, линейные графики и диаграммы рассеяния).
  • настраивать внешний вид графиков (например, цвета и шрифты).

About this course

Цели курса.

Курс предназначен для тех, кто хочет быстро начать учиться и использовать библиотеки Pandas, Numpy и Matplotlib. В ходе курса вы научитесь работать с данными, выполнять вычисления и визуализировать данные.

Почему стоит выбрать именно этот курс?

Курс собрал в себе сразу три популярные библиотеки для работы с данными и научных вычислений. Информация на курсе очень хорошо структурирована и позволяет быстро начать учиться. Вы также получите необходимые знания для дальнейшего изучения Python.

Что приобретут учащиеся после успешного освоения курса?

После успешного освоения курса вы научитесь работать с данными, выполнять вычисления и визуализировать данные.

Особенности курса.

Курс предоставляет удобно собранную информацию воедино для работы с данными и мощные инструменты для научных вычислений. Курс также предоставляет возможность работать с многомерными массивами данных и сохранять графики в различных форматах.

В ходе курса вам нужно будет выполнять задания по работе с данными и визуализации данных.

В каждом разделе вы будете выполнять практические задания по работе с соответствующими инструментами.
